# **App Name**: ScholarAI

## Core Features:

- Content Extraction: Extract text from documents, images, text input, and URLs for analysis.
- AI Study Guide Generation: Generate a study guide using a summarization and information extraction tool based on uploaded content. This feature will summarize content into key concepts and provide clear explanations.
- AI Assessment Generation: Create tests (multiple choice, flashcards, oral questions, essay questions) from the uploaded materials using AI to generate relevant and challenging questions.
- Customizable Test Parameters: Allow users to customize tests by setting the number of questions, the passing score and test timer.
- Performance Tracking and Feedback: Provide a test results page that has the score, a performance summary, and tailored recommendations for improvement.
- Persistent Data Storage: Save all study materials and completed tests for future access in Firebase.
- User Account Management: Allow users to create accounts and log in using email or Google (Firebase Authentication), and show recent material on their dashboard.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence and focus.
- Background color: Very light blue (#E8EAF6), almost white, providing a gentle contrast that does not distract from the content.
- Accent color: A violet shade (#7E57C2), a close analogue of the primary color with shifted brightness and saturation, for highlighting key interactive elements without overwhelming the user.
- Body and headline font: 'Inter', a sans-serif typeface offering clarity and a contemporary feel.
- Use clear, minimalist icons to represent different types of study materials and test formats.
- Design a clean, intuitive dashboard layout that allows users to easily access study materials, past tests, and progress tracking.
- Incorporate subtle transitions and loading animations to enhance the user experience and provide feedback during content processing.
